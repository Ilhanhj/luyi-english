# **App Name**: LuyiEnglish Website

## Core Features:

- Hero Section with CTAs: Eye-catching hero section with headline, subheadline, primary and secondary CTAs (Daftar Gratis, Lihat Tutor) and trust bar.
- Tutors Carousel: Responsive carousel displaying tutor information: photo, name, specialization, rating, short bio, review quotes, and a 'Book Trial' button. The Tutor list comes from a Firestore database.
- Testimonials Carousel: Carousel displaying WhatsApp screenshot mockups of testimonials, with a summary text and name initials, and a lightbox option to view the full screenshot. Testimonials list data from Firestore.
- Pricing Section: Display three pricing packages (Basic, Regular, Intensif) with features comparison, price, and a 'Pilih Paket' CTA for each.
- FAQ Accordion: Accordion component with frequently asked questions and answers.
- Contact & Signup Form: Contact form that allows users to sign up for classes. When submitted the information is stored on a Firestore database.

## Style Guidelines:

- Primary color: #3B82F6 (Blue), chosen to convey trust and professionalism.
- Background color: #F7FAFC (Light Gray), provides a clean and subtle backdrop.
- Accent color: #EC4899 (Pink), complementing the blue for a slightly playful and modern touch.
- Font: 'Inter', a versatile sans-serif for headlines and body text, ensures readability and a modern feel.
- Use clean, modern icons that align with the brand's friendly and professional tone.
- Implement a responsive, mobile-first layout using Tailwind CSS, ensuring optimal viewing experience across devices.
- Subtle fade and slide animations using AOS for smooth transitions on scroll, with a duration of 400-700ms.
